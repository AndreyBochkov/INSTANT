Aplikasi dan infrastruktur INSTANT telah diluncurkan dalam mode uji coba. Perubahan signifikan dapat dilakukan pada protokol keamanan dan mekanisme pertukaran data. Saya tidak akan memperkirakan durasi periode pengujian. Terima kasih telah membaca.

Apa itu?

Ini adalah utusan yang berfokus pada keselamatan dan kecepatan. Kurangnya pelacak, pengumpul data, dan algoritma enkripsi modern meningkatkan kualitas-kualitas ini. Proyek ini dikembangkan dan diluncurkan oleh satu orang dalam kerangka kerja individu.

Apa tujuan dari INSTANT?

Hal ini segera perlu untuk menjelaskan bahwa ini adalah proyek non-komersial. INSTANT tidak bertujuan untuk menyediakan Anda dengan iklan yang dipersonalisasi, tidak bertujuan untuk mengumpulkan uang. INSTANT ada di sini untuk pertukaran aman byte antara dua perangkat yang terhubung ke Web.

Keamanan tanpa HTTPS? Aku bisa melihat, "http://i..."

Memang, INSTANT tidak menggunakan HTTPS di situs. Namun, aplikasi menggunakan protokol enkripsi khusus berdasarkan algoritma Diffie-Hellman yang disederhanakan dari TLS 1.3, dan sehingga bekerja sangat cepat. Secara sederhana, TLS melakukan lebih banyak untuk kompatibilitas dengan perangkat apa pun, dan protokol INSTANT hanya berfungsi di antara ponsel Anda dan server itu, sehingga dapat mengatur koneksi untuk sejumlah kecil paket.

Semua orang mengumpulkan data tentang semua orang, itu sudah jelas.

Tapi bukan utusan pribadi. Ya, ini hanya untuk Android, tetapi untuk iOS sudah ada "bitchat" yang menggabungkan pengguna ke jaringan terdesentralisasi (tanpa server), menjamin anonimitas. INSTANT memastikan kinerja - berkat sentralisasi. Berapa banyak startup, begitu banyak pendekatan. Tetapi menguntungkan untuk mengumpulkan data untuk pengiklan, bukan programmer.

Kebijakan Privasi dan Ketentuan Penggunaan?

Ya, mereka, setransparan dan sesederhana mungkin. Mereka berada di situs yang sama. Perlu diingat bahwa setiap penggunaan aplikasi seluler melibatkan membaca, memahami semua poin dan setuju dengan mereka.

Jadi, saya ingin mengunduh messenger.

Aplikasi INSTANT didistribusikan "sebagaimana adanya", secara bebas untuk digunakan. Ia berada di situs yang sama. Buka file di ponsel Anda dan instal aplikasi.

Keterbatasan teknis?

Sayangnya, ada. INSTANT tidak bernegosiasi dengan produsen smartphone tentang resolusi absolut aktivitas latar belakang. INSTANT tidak menggunakan teknologi Google Firebase Cloud Messaging untuk menghindari berbagi data dengan server pihak ketiga. Karena itu, ketika layar dimatikan, pesan tidak disampaikan. Sejauh ini, INSTANT lebih tentang keselamatan daripada tentang kecepatan, tetapi pekerjaan sedang dilakukan untuk meningkatkan.
